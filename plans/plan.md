# План разработки лендинга для курса по логопедии для детей

## Сводка требований

| Параметр       | Значение                                    |
| -------------- | ------------------------------------------- |
| Тип проекта    | Лендинг (одностраничный сайт)               |
| Тематика       | Образовательный курс по логопедии для детей |
| Структура      | 5 блоков                                    |
| Цветовая схема | Профессиональные тона (синий, белый, серый) |
| Контакты       | Форма обратной связи                        |

## Структура лендинга

```mermaid
block-beta
    columns 1
    block[Hero]:1
    block[О курсе]:1
    block[Программа]:1
    block[Отзывы]:1
    block[Контакты]:1
```

### Блок 1: Hero (Приветствие)

- **Заголовок**: "Профессиональный курс по логопедии для детей"
- **Подзаголовок**: краткое описание преимуществ курса
- **CTA-кнопка**: "Записаться на курс"
- **Фон**: градиент синего цвета
- **Элементы декора**: плавные волны внизу блока

### Блок 2: О курсе

- **Заголовок**: "О нашем курсе"
- **Текст**: описание курса, его целей и преимуществ
- **Преимущества**: 3-4 карточки с иконками
  - Индивидуальный подход
  - Опытные специалисты
  - Игровая методика
  - Результат гарантирован

### Блок 3: Программа курса

- **Заголовок**: "Программа обучения"
- **Структура**: 4-5 модулей курса
  - Модуль 1: Диагностика речевых нарушений
  - Модуль 2: Артикуляционная гимнастика
  - Модуль 3: Постановка звуков
  - Модуль 4: Развитие связной речи
  - Модуль 5: Работа с родителями
- **Формат**: аккордеон или вертикальный список с описанием

### Блок 4: Отзывы

- **Заголовок**: "Отзывы наших учеников"
- **Карточки отзывов**: 3 отзыва с фото, именем и текстом
- **Навигация**: стрелки для переключения (если отзывов больше 3)
- **Дизайн**: карточки с лёгкой тенью и скруглёнными углами

### Блок 5: Контакты

- **Заголовок**: "Свяжитесь с нами"
- **Форма обратной связи**:
  - Поле "Имя" (текстовое)
  - Поле "Телефон" (телефонное)
  - Поле "Email" (email)
  - Поле "Сообщение" (textarea)
  - Кнопка "Отправить"
- **Дополнительно**: контактная информация (телефон, email)
- **Валидация**: проверка обязательных полей на клиенте

## Технические решения

### Структура файлов

```
/
├── index.html          # Основной HTML-файл
├── css/
│   └── styles.css      # Стили (встроены в HTML для простоты)
└── js/
    └── main.js         # JavaScript (встроен в HTML)
```

### Цветовая палитра

- Основной синий: `#2563eb` (blue-600)
- Тёмно-синий: `#1e40af` (blue-800)
- Светло-синий: `#dbeafe` (blue-100)
- Белый: `#ffffff`
- Серый текст: `#4b5563` (gray-600)
- Светло-серый фон: `#f9fafb` (gray-50)

### Типографика

- Заголовки: 2.5rem - 3rem (адаптивно)
- Подзаголовки: 1.25rem - 1.5rem
- Основной текст: 1rem
- Шрифты: системный стек (system-ui, -apple-system, sans-serif)

### Адаптивность

- Мобильные устройства: < 768px
- Планшеты: 768px - 1024px
- Десктоп: > 1024px

### Интерактивность (JavaScript)

- Плавная прокрутка при клике на навигацию
- Валидация формы обратной связи
- Анимация появления элементов при скролле
- Мобильное меню (если будет навигация)

## Порядок реализации

1. **HTML-разметка** — создание структуры всех 5 блоков
2. **CSS-стилизация** — оформление каждого блока
3. **Адаптивность** — медиа-запросы для разных устройств
4. **JavaScript** — добавление интерактивности
5. **Тестирование** — проверка отображения и функциональности
